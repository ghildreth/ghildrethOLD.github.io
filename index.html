Passion

I've found beauty in ideas that transcend Computer Science, namely: Algorithms, Logarithms and Recursion. Here's an example of a function that, so simply, displays the elegant nature of recursion in terms of mathematics.

function power(base, exponent) {
  if (exponent == 0) {
    return 1;
  } else {
    return base * power (base, exponent - 1);
  }
}
console.log(power(2, 3));
